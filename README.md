<h2>Description</h2>
<p>Tic-tac-toe is a paper-and-pencil game for two players, X and O, who take turns marking the spaces in a 3Ã—3 grid. The player who succeeds in placing three of their marks in a horizontal, vertical, or diagonal row is the winner.

More info about the game here -> <a href="https://en.wikipedia.org/wiki/Tic-tac-toe">Wikipedia</a></p>

<h2>How to play</h2>
<p>Click in an empty cell and beat the CPU before time runs out.</p>
